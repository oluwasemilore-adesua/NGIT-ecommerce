1. A computer is an electronic device that accepts data(input), processes according to instructions, and produces useful information(output). 

Difference between Hardware and Software
Hardware refers to the physical parts of a computer, the components you can see and touch. Examples: Keyboard, monitor, and the central processing unit(CPU). 

Software refers to the programs and instructions that tell the hardware what to do. Software cannot be touched. Examples: Windows Operating System, Microsoft word, Google chrome. 

2. An operating system is a software that manages a computer's hardware and software resources and provides a platform for other programs to run. It acts like a bridge between the user and the computer hardware. 

Two Primary Functions
It manages the computer's hardware like the CPU, memory, and storage to make sure programs run efficiently. 
It provides the user interface (UI) that allows users to interact with the computer. 

3. Windows is a widely used operating system for personal computers. It is known for its user friendly interface and broad hardware and software compatibility, which makes it easy for beginners to navigate. 

Linux is an open-source operating system, which means it is mostly free and its source code can be modified by anyone, and its known for its stability, flexibility and security, powering everything from desktops to the majority of the world's servers.

Differences
Windows usually requires a license, while Linux is mostly free. 
Windows focuses on ease of use with a GUI, while Linux offers flexibility with GUI or command-line options. 
Windows supports most commercial software, while Linux relies more on open-source alternatives.
 
4. File: A file is a collection of data or information stored on a computer under a specific name. it can contain text, images, videos, music or programs. Files are used to store information so it can be opened, edited, or shared later. Example: assignment.txt, photo.jpg. 

Folder(Directory): A folder, also called a directory, is a container used to organize and store files on a computer. It helps keep files arranged neatly and makes them easy to find. A folder can also contain other folders(subfolders). 

File Extension: A file extension is the ending part of a file name, usually after a dot(.). (e.g., .pdf, .jpg, .exe). It tells the computer what type of file it is and which program should open it. 

5. The command line is a text-based interface that allows users to interact with a computer by typing commands instead of clicking icons or buttons. Through the command line, users can run programs, manage files and folders, install software, and control system operations directly. 

Why it is important for developers: The command line is important for developers because it allows them to work faster and more efficiently. Developers use it to run code, manage projects, and use tools like Git for version control. It also makes it easier to automate tasks, troubleshoot problems, and work on servers or development environments that may not have a graphical interface.    

6. Server: A server is a computer system that stores information and provides services or data to other computers(called clients) over the internet. For example, when you open a website, the server sends the website data to your device. 

Browser: A browser is a software application used to access and view websites on the internet. It allows users to retrieve information from servers and display it in a readable format. Examples: Google chrome, Microsoft edge. 

IP Address: An IP(Internet Protocol) address is a unique numerical identifier assigned to every device connected to the internet. It helps computers locate and communicate with each other online. Example: 192.168.1.10.

DNS(Domain Name System): DNS is like the internet's phonebook. It converts data easy-to-remember website names(like www.google.com)into IP addresses that computers use to find servers. 

HTTP and HTTPS:
HTTP(Hypertext Transfer Protocol) is used to transfer data between a browser and a server, but it is not secure. 
HTTPS(Hypertext Transfer Protocol Secure) is a secure version of HTTP. It encrypts data, making it safer from hackers. HTTPS is more secure than HTTP because it protects data during transmission.

7. The client-server model is a way computers communicate over a network where one computer(the client) requests a service or information, and another computer(the server) provides it. 
Example: When you open a website using a browser like google chrome, your computer or phone acts as the client. You type a website address(for example, www.example.com) and press enter. The browser sends a request to the server where the website is stored. The server then processes the request and sends the web page data back to your browser, which displays the website on your screen. In this model, the client makes the requests, and the server responds by supplying the requested information or service. 

8. Frontend Development deals with the user-facing part of a website or application-everything users see and interact with directly. This includes the layout, design, buttons, forms, colors, and text. Frontend developers focus on making the website appealing, responsive and easy to use. Common frontend technologies include html, CSS, and JavaScript.

Backend Development focuses on the behind-the-scenes functionality of a website. It handles how the website works, processes user requests, manages databases, and ensures data is stored and retrieved correctly. Backend developers work on servers, databases, and application logic. Common backend technologies include Python, PHP, Java, Node.js, and databases like MySQL. 

9. A Website is a collection of webpages that contain information such as text, images, videos, or services that users can access through the internet. Examples include blogs, online stores, and school websites. 

A Domain is the unique address of a website that people type into a browser to access it. It makes websites easy to find instead of using long IP addresses. Example: www.google.com.

Hosting is a service that provides storage space on a server where a website's files are kept. Hosting allows the website to be available on the internet 24/7. 

How they work together: The website contains the content, the domain is the address used to find the website, and hosting stores the website files on a server. When a user types a domain name into a browser the browser locates the hosting server and displays the website.  

10. The Browser Rendering Process is how a web browser turns a website code into what you see on the screen. 
First, the browser requests the webpage from a server and downloads the HTML, CSS, and JavaScript files. Next, it reads the HTML to build the structure of the page and applies the CSS to style it(colors, fonts, layouts). Then the browser executes JavaScript, which can change content or add interactivity. Finally, the browser lays out and paints everything on the screen so the webpage appears visually to the user.
