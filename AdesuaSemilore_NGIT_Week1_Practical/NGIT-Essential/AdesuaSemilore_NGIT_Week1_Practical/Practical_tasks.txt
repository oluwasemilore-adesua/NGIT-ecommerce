Task 4: Internet and Web Observation

Website visited: www.wikipedia.org

Domain name: wikipedia.org

Protocol used: HTTPS

What happens behind the scenes when I press Enter in the browser:

When I press Enter, the browser first sends a request to the DNS to find the IP address of the website. Once the IP address is found, the browser connects securely to the server using HTTPS. The server then processes the request and sends back the website files such as HTML, CSS, and images. The browser receives these files, processes them, and displays the webpage on the screen.
